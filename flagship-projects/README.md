# Flagship Projects

Flagship projects are the most serious systems in the Super ecosystem.

They are intended to become expert-reviewable proof-of-work projects.

---

## Expected Standard

Flagship projects must exceed the Phase 0 Toolkit Standard.

They should also follow:

- Core-Authentic Real-World Project Standard
- Data Authenticity Standard where data is involved
- Real-world use-case framing
- Data-source disclosure
- Validation checks
- Assumption tracking
- Limitations section
- Reports or user-facing outputs
- Readiness levels
- Future upgrade path

---
