## Dependency Protection

The existing Financial Data Pipeline, India + US Dashboard, AI Research Copilot, Macro Dashboard, Cross-Market Comparator and other systems are not replaced. GMP projects provide reusable shared architecture and validated reference implementations that these existing systems consume.

## Registry Rule

A project enters this formal count only when it is a named phase project, capstone, flagship, supporting system or explicitly approved major deliverable. Small exercises and learning simulations remain outside the major-project total.
