# Supporting Projects

Supporting projects strengthen the flagship systems.

They may be smaller than flagship projects, but they should still be clean, useful, tested, and documented.

---

## Expected Standard

Supporting projects should generally follow the Phase 0 Toolkit Standard unless intentionally marked experimental.

---

## Expected Supporting Projects

- Portfolio Cockpit
- Sovereign Risk Dashboard
- Options and Volatility Lab
- Market Microstructure Toolkit
- Cross-Market Capital Growth Comparator
- Stock X-Ray / Fundamental Analyzer
- RegTech / Surveillance Module
