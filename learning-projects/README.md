# Learning Projects

Learning projects are smaller projects used to master specific engineering concepts.

They do not always need full flagship-level infrastructure, but they should still be clean and GitHub-ready where useful.

---

## Minimum Standard

A learning project should usually include:

- Clear folder structure
- README
- Runnable examples
- Tests where useful
- Notes or explanation where useful

---

## Example Learning Projects

- Circuit simulators
- Resistor-network calculators
- Logic-gate simulators
- CPU/cache experiments
- Mini shell
- Networking tools
- Data-structure libraries
- Small ML tools
- Embedded prototypes
- Visualization utilities
