# Progress

This file tracks major completed milestones in the Super ecosystem.

This is not a daily study log.  
It should be updated only when a major roadmap phase, capstone, professionalization sprint, flagship project, or important repository milestone is completed.

---

## Progress Tracking Rule

Update this file only when one of the following happens:

- A full phase is completed
- A capstone project is completed
- A project is professionalized
- A major project repository is created
- A Tier 1 or Tier 2 project reaches a readiness level
- A major roadmap standard is created or changed
- A major milestone is pushed to GitHub

Small topic-level progress does not need to be tracked here.
